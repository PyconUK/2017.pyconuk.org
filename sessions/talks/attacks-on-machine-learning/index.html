Attacks on Machine Learning

prabhant singh | Friday 17:00 | Room C

The session is about adversarial machine learning. The talk will include 1.Introduction to adversarial machine learning 2.Crafting adversarial examples with various algorithms. 3.Properties of adversarial examples 4.Attacks on CNN and NLP systems with adversarial examples 5.Defenses against adversarial examples.
